### Eclipse 유용한 단축키

##### Source 편집

| 단축키           | 의미                      |
| ---------------- | ------------------------- |
| Ctrl + Shift + O | 자동 import               |
| Ctrl + Space     | 입력 중 자동 완성 기능    |
| F2               | 컴파일 에러 원인 보여주기 |
| Ctrl + i         | 자동 들여쓰기             |
